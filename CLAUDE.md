# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Purpose

Parser/extractor for **Plataforma Brasil** (Brazilian Ministry of Health research ethics platform). Extracts structured data from HTML pages exported from the platform, focusing on CEP/CONEP project appraisals (`detalharProjetoAgrupadorApreciacao`).

## Reference HTML

`caae_exemp.html` — example page exported from Plataforma Brasil. Use as reference for HTML structure when writing parsers. Key elements:

- Form action: `/visao/administrador/4x4Novo/detalharProjetoAgrupadorApreciacao.jsf`
- User role displayed in `#detalheUsuario`
- Project/appraisal data in `<table>` elements with RichFaces components
- CAAE field present in page body
- Modal panels via RichFaces (`ajaxStatusMP`)
- JSF ViewState: `javax.faces.ViewState`

## HTML Quirks

- Generated by JSF + RichFaces 3.3.3 — heavy use of `j_id*` auto-generated IDs
- `detalharJustificativa*` elements are RichFaces modal panel fragments
- Page uses `jQuery.noConflict()` — jQuery available as `jQuery`, not `$`
- Charset: UTF-8, but some accented chars encoded as HTML entities (`&ccedil;`, `&atilde;`)
